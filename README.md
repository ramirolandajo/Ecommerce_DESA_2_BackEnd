# 🛒 Web de Compras

Este módulo representa la **interfaz de e-commerce** para los usuarios finales.  
Gestiona usuarios, carritos, órdenes y pagos.

## 🚀 Funcionalidades
- Registro y autenticación de usuarios.
- Carrito de compras.
- Órdenes de compra.
- Checkout y pagos.
- Administración básica de usuarios.
